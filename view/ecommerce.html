Qu'est-ce que le projet E-commerce ?

Le projet E-commerce est un projet réalisé en trinôme durant le semestre 3 de BUT informatique.

L'objectif de ce projet était de réalisé un site internet de E-commerce en priorisant l'utilisation des langages PHP et SQL via PDO.

Ce projet intègre une gestion de comptes utilisateurs et aministrateurs permettant l'ajout/surpression de produits, une gestion de panier et de commandes.

Technologies utilisés :

Compétences mobilisés :

  • Utilisation du model MVC

  • Mise en place d'un routeur PHP

  • Utilisation de sessions et cookies

  • Création d'une base de données

  • Utilisation de procédures stockés

  • Intéraction avec une base de données via PDO

  • Utilisation de requêtes préparées

  • Travail d'équipe
